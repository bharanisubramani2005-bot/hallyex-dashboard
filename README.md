# 🥤 NutriShake – Full Stack Smoothie Ordering & Admin Dashboard

A modern full-stack web application for ordering smoothies and managing operations through an advanced admin dashboard.

---

## 🚀 Features

### 👤 User Side

* Browse smoothie menu 🍓🥭
* Add to cart with animated "+" button ➕✔️
* Place orders easily
* User profile with avatar selection 🐰🐼
* Auto-fill customer details from profile
* Responsive UI (Mobile / Tablet / Desktop)

---

### 🛠️ Admin Dashboard

* Secure Admin Login 🔐
* Menu Management (Add / Edit / Delete items)
* Customer Orders Table 📋
* Order Status Tracking:

  * Pending
  * In Progress
  * Completed
* Three-dot menu (⋮) for Edit/Delete actions
* Create Order Popup Form
* Auto-fetch customer info from user profile
* Indian currency support (₹)

---

### 📊 Dashboard & Analytics

* KPI Cards:

  * Total Revenue 💰
  * Total Orders 📦
  * Average Order Value
* Charts:

  * Sales by Product 📊
  * Daily Sales 📈
  * Status Distribution (Pie Chart)
* Custom Dashboard Builder:

  * Drag & Drop Widgets
  * Bar / Line / Pie Charts
  * Table & KPI widgets

---

### ⚙️ Settings & Profile

* User profile with avatar selection
* Save user details (localStorage)
* Auto-sync with order forms
* Clean UI with smooth animations

---
